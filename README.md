## **Installation**

- Clone Repositories

```
https://github.com/luthfidiqi/Tickie-Frontend.git
```

- Open Project Folder

```
$ cd Tickie-Frontend
```

- Intall NPM Package

```
npm install
```

- Set up .env file

```
REACT_APP_URL_BE=
REACT_APP_URL_CLOUDINARY=
```

- Start The Project

```
npm start
```
